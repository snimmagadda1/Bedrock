# Center

## Use case

The Center is designed to center and clamp the width at the a defined width, the medium breakpoint by Default

## How to install

If you haven't already, install `styled-components`:

```bash
  npm install styled-components
```

Then install the stack component

```bash
  npm install @bedrock-layout/center
```
